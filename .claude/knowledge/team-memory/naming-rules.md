# 강팀 작명 룰 — 짧고 임팩트 있게

> 아이템·기능·캠페인·프로젝트에 이름을 지을 때. **길고 나열식 금지.** 부르기 좋고, 기억에 남고, 짧게. 재미있는 줄임말 환영.

---

## 1. 원칙 — 4가지

1. **짧게** — 2~4글자 목표. 길어도 한 호흡에 부를 수 있게.
2. **부르기 좋게** — 발음 굴러가야. 입에 안 붙으면 탈락.
3. **기억에 남게** — 한 번 들으면 안 잊히는 후크. 말장난·라임·의외성.
4. **임팩트** — 나열("사용자 합격 사례 공유 게시판")이 아니라 *한 방*("합격각").

---

## 2. 안티패턴 — 이렇게 지으면 별로

| ❌ 나쁜 예 | 왜 | ✅ 고친 예 |
|---|---|---|
| "사용자 활동 기반 자산 적립 시스템" | 길고 설명식 | **"쌓기"** 또는 **"적립각"** |
| "강사 자료 공유 커뮤니티 플랫폼" | 나열, 임팩트 0 | **"강자료"** (강사 자료) / **"쌤빵"** |
| "합격 사례 포트폴리오 게시판" | 관공서 문서 톤 | **"합격각"** / **"붙자"** |
| "AI 자동 회의록 생성 및 전송 기능" | 기능 나열 | **"회의뚝딱"** / **"톡회의"** |

→ *무엇을 하는지* 다 담으려 하지 말 것. *느낌* 하나만 잡아라.

---

## 3. 작명 기법 (하나 이상 써라)

### ① 줄임말 (가장 강력)
- 핵심 단어 앞글자·핵심 음절만. "강사 자료" → **강자료**, "합격 사례" → **합사** / **합격각**
- 재미 요소 넣기: 발음이 웃기거나 의외면 +점

### ② 라임·반복
- "톡톡", "쌓쌓", "뚝딱" — 반복음은 기억에 박힘

### ③ 의외의 조합
- 익숙한 단어 + 살짝 비틀기. "합격" + "각(느낌)" = **합격각** (요즘 말투)

### ④ 동사화·구어체
- 명사 나열 대신 *행동*. "붙자", "쌓아", "떠라" — 입에 붙는 구호

### ⑤ 브랜드 접두사 재활용
- 강팀 계열이면 "강-" 접두사 (강자료·강체크). 단 남발 금지.

---

## 4. 회의에서 작명할 때 — 절차

아이템 이름이 필요하면 **아뱅이 주도** (구어·심리 감각). 절차:

1. **후보 3개** — 각 다른 기법. 나열식 1개 (비교용) + 짧은 임팩트 2개.
2. **각 후보 한 줄 평** — 왜 이게 부르기 좋은지 / 걸리는 점.
3. **강대표가 픽** — 짧고 묵직하게 "○○로 가자."
4. **전원 반응** — 별로면 반대. (모두 찬성 금지 룰 적용)

### 예시 대사

```
아뱅: 이 합격 사례 게시판, 이름 세 개 뽑아봤어.
  ① "합격 사례 공유 게시판" — 정직한데 노잼, 아무도 기억 못 함.
  ② "합격각" — 요즘 말투, 짧고 "나도 붙겠다" 느낌 팍.
  ③ "붙자" — 구호형, 커뮤니티 구호로 딱.
  (②③ 밀고 싶다. 판 흔드는 맛.)
강체크: "합격각"은 좋은데, 검색 노출은 "합격 사례"가 유리해요. (SEO 걱정)
아뱅: 오 맞말. 그럼 표시는 "합격각", 메타태그·title은 "합격 사례"로 이원화. 어때?
강대표: 합격각. 가자. (짧게 끝냄)
```

## 5. 금기

- **기능 전부 담기** — 이름은 요약이 아니라 후크.
- **영어 남발** — "MyPortfolioHub" 류. 한국 사용자면 한글 구어가 더 박힌다 (단 글로벌 타겟이면 영어 OK).
- **모두 찬성** — 이름도 회의. 최소 1명 딴지 (보통 강체크 = 검색·중복·상표).
- **강대표가 길게 설명** — 픽은 짧게.
