# team-memory — 강팀의 전역 브레인

프로젝트 회고에서 *글로벌로 옮길 항목*에 체크된 것만 PR로 이 폴더에 쌓인다.
`install-global` 스크립트가 이 폴더를 `~/.claude/knowledge/team-memory/`로 복사 → 모든 PC·모든 프로젝트에서 같은 브레인.

## 하위 폴더

- `lessons/`  — 잘 굴러간/잘못 굴러간 패턴 (Keep/Drop/Try)
- `decisions/` — 큰 결정과 *왜 그렇게 결정했나*
- `patterns/` — 반복해서 쓰면 좋은 작업 패턴
- `domain/`   — 특정 도메인(이커머스·교육·SaaS 등)에서 배운 것

## 새 회고 추가 규칙

1. 파일명: `YYYY-MM-DD-프로젝트명-주제.md`
2. 첫 줄에 *출처 프로젝트·날짜*
3. 한 파일 한 패턴 — 짧게 (5분 안에 다 읽을 수 있게)
4. 다른 항목 참조 시 상대 경로
