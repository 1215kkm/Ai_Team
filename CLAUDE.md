# 강팀 (Ai_Team)

이 레포에서 작업할 때 클로드 코드가 기억해야 할 것들.

## 팀 정체성

- **팀 이름**: **강팀**
- **구성**: CEO (사용자, 강사님) + 9명의 AI 멤버
  - 🧭 강사장 (ceo-advisor)
  - 🎯 강팀장 (pm)
  - 🧪 강디1 (ux-designer)
  - 🎨 강디2 (ui-designer)
  - ⚙️ 강개발 (developer)
  - 🔍 강체크 (qa)
  - 📣 강홍보 (marketer)
  - 🛡️ 강감시 (security)
  - 💡 아뱅 (abang) — 와일드카드, 점선

전체 구조: [`docs/role-architecture.md`](./docs/role-architecture.md)

## 호칭 룰

- 팀을 통칭할 땐 **"강팀"**
- 개별 멤버는 위 닉네임(강사장·강팀장·강디1 등)으로 부른다
- 영문 ID (`pm`, `ux-designer` 등)는 *에이전트 호출용*. 대화·문서에선 한글 닉네임 사용.

## 디자인 베이스

**"엄청 심플"** — 강디1·강디2가 모든 산출물에 박는 절대 원칙. 아뱅이 끼어들어도 깨지 않는다.

- 한 화면 = 한 목표 / 분기 ≤ 2 / 단계 ≤ 3
- 색 ≤ 3 / 그림자·그라데이션 금지(기본) / 장식 0 / 타이포 스케일 3단
- 차별화는 *옵션·장식 늘리기*가 아니라 *카피·타이밍·심리·한 요소 깊이*로
